入门级操作系统分析科普

发表于 2020-05-16 | 分类于 攻略

入门级操作系统分析介绍

Windows · Linux · Mac OS

前言

对于大部分电脑用户来说，操作系统是每天都要和其打交道的东西，但是很多人对操作系统本身并不了解，甚至可以用无知来形容。其实，对于电脑用户，尤其是计算机办公的用户来说操作系统和工作效率有很大的关系，一款合适的操作系统会大大提高用户的工作效率。

就目前全球桌面级操作系统消费市场来说主要是以Windows、Linux和Mac OS这三家操作系统为主，形成三组鼎立的局面。对于大部分人来说，Windows绝对是首选的操作系统，也有不少人会选择高端大气上档次的Mac OS，Linux虽然不及上面两家但是由于其发行版众多但也受到小部分用户的青睐。

如何选择操作系统，就要从实际出发，仔细分析自己的使用需求，在进行选择。

Windows

介绍

Windows操作系统相信大家并不陌生，它诞生于1985年，从最早的Windows 1.0到现在的Windows 10，微软公司已经为我们提供了40多年的操作系统服务。作为目前全球桌面级系统市场份额第一的操作系统，Windows 10有着其它系统无法比拟的庞大生态圈和用户基数，同时有着最好的硬件兼容性和驱动支持。

Windows从最开始的1.0发展到现在的10，期间整个系统的底层其实是由翻天覆地的变化的。

历史上最受好评的操作系统莫过于Windows XP，作为一款超现象级操作系统，XP创造了世界操作系统使用量的历史记录，直至今日，仍有很多用户仍然在使用Windows XP。但是XP系统自2014年微软停止其技术支持后，它的软硬件兼容性越来越差，很多软件开发商也停止了对Windows XP的支持。

再后来虽然有过Windows Vista，但其因为异常辣鸡的稳定性饱受诟病，昙花一现过后便悄然淡出了操作系统的市场。

微软历史上最成功的操作系统Windows 7可谓是让微软赚的盆满钵满，Windows 7以优异的用户体验迅速收获了7大批市场份额。Windows 7刚刚发售时也因为系统稳定性的问题被挂墙批判了好一段时间，但后续微软的优化做得相当到位，直至今日Windows 7依然是Windows系列最稳定的操作系统。

但是从7以后，微软莫名其妙的走了歪路。Windows 8和Windows 8.1的使用体验虽不及Vista那么辣鸡，但也好不到哪里去，以至于这两款操作系统很快就被用户抛弃了，那段时间微软虽然也进行了很多尝试，甚至想要进军手机市场，但结果并不尽人意，直至Windows 10的出现。

2014年10月1日，微软在旧金山召开新品发布会，对外展示了新一代Windows操作系统，将它命名为“Windows 10”，并宣布这是Windows系列最后一款操作系统，今后并不会再开发新的系统而是在Windows 10的基础上迭代更新。

优点

办公

办公行业的从业人员都很清楚Windows的办公效率，Office全家桶（Word, PPT, Excel 等）个个都是顶尖的办公软件，同时支持Android移动平台，有着不错的兼容性。其次还有Onedrive云盘，To Do等高效的办公软件，也为Windows的高效办公加了不少分。

大众娱乐

Windows操作系统上支持安装各种媒体播放器和网络浏览器，一般用户追剧、看番、刷微博的功能都可以满足。确切的说，正式Windows的普及满足了用户的娱乐需求。

同时，Windows良好的硬件支持使得某些发烧友可以尽可能的享受诸如HIFI、4K等功能，以及硬件爱好者可以全凭自己的主观喜好对电脑进行升级而不用管操作系统的兼容文体。

游戏

经常玩大型游戏的用户都清楚Windows系统对于游戏的重要性。从开发上来说，很多游戏都是基于虚幻4、Unity等引擎进行开发的，这些游戏大部分都不会去做其他平台的移植，因此单从数量上说，Windows要远远优于其他桌面操作系统（不包括PS、Wii等）。而Windows的硬件支持也保证了驱动的稳定性，为游戏画面效果和流畅度提供了稳定的保障。

创作

Windows为艺术、影视创作者提供了良好的创作环境（Adobe等）

缺点

系统稳定性

Windows众多版本中，Windows 7的稳定性最好，但是微软已经停止了对它的技术支持，不过硬是要用的话也还是可以使用的。
